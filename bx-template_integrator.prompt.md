Пространство имен: ${input:namespace:bitrix}
Имя компонента: ${input:componentName}
Имя шаблона: ${input:templateName:.default}

Корень проекта: определи из результата команды `pwd` в терминале перед началом работы.

При записи/перезаписи файлов предпочитай shell-команды (heredoc через cat) вместо инструментов редактора VS Code, чтобы избежать несогласованности между виртуальным слоем редактора и реальной файловой системой.

Найди дефолтный шаблон компонента по пути:
`local/templates/.default/components/${namespace}/${componentName}/${templateName}/`

Интегрируй HTML-верстку из файлов `*.html` в одноимённые `*.php` файлы шаблона (например `template.html` → `template.php`):
- Замени существующую HTML-структуру на верстку из `*.html` файла.
- Сохраняй весь PHP-код и его функциональность без изменений.
- Сохраняй функциональные HTML-элементы из `*.php` файла: `<form>`, `<input>`, `<select>`, `<button>` и их атрибуты — интегрируй их в новую верстку в соответствующие места.
- Если есть `*.js` файлы — проверь используемые в них id и классы, добавь их в интегрируемую верстку чтобы сохранить функциональность JS.
- Файлы `*.html` не удаляий и не изменяй.

После завершения сообщи только если возникли ошибки.
